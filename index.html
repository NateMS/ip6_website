UML4VIP

helping visually impaired people create UML class diagrams

About this project

As part of a Bachelor's thesis we wanted to make UML class diagrams more accessible to visually impaired people. Over the course of several months and with the help of multiple blind software engineers we developed a prototype application that offers unique advantages compared to more traditional methods. The application works with a 3D printed grid that is places part of the screen which helps navigate the various classes in a coordinate system.

The iOS App

The iOS app is compatible with iOS 13 and higher and is available on all iPads. The application can be downloaded on the App Store.

In order to use the application to its full potential, we recommend that you connect an external keyboard to the device. The app has been designed to work well with the voiceover functionality of iOS.

Features

While the app is not yet feature complete, a wide range of interactions are already available which make the app usable for real work:

  • create new diagrams
  • create, edit and delete:
    • classes and interfaces (up to 36 classes)
    • fields and methods for classes
    • relationships between two classes
  • support for various levels of visibility (public, protected, private)
  • easily read information, such as relationships, fields and methods
  • multiple modes which allow for easier interactions
  • move classes around the grid as you wish
  • export the diagram either as an image (png) or as a .puml file (PlantUML)

3D printed Grid

You need a 3D printed custom grid which fits on your iPad so you can use the application as intended. You can download the file for the 3D printed grid here:
